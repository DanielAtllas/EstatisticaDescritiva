Introdução - Estatística Descritiva

A estatística em sí, se dá na dependência de dados com o objetivo de de permitir uma interpretação e análise dos mesmos, ou seja, é a capacidade de interpretar dados após a sua coleta, a fim de organiza-los e compartilha-los atrvés de gráficos, tabelas e etc. De certa forma, este conteúdo está muito relacionado aos conceitos de BI e Big Data, onde uma coleta de dados é realizada de maneira a trabalhar esses dados e deveolve-los, gerando assim novos dados para análise. Logo, pode ser resumido em etapas de:

  1. Coleta

  2. Análise/Interpretação

  3. Exibição

Diante disso, são realizadas técnicas de coleta e amostragem de dados para a realização dessas estatísticas. Além disso, esses dados possuem uma veracidade temporária (com possibilidade de não serem válidos após um determinado tempo desde sua coleta), assim como a variação dessa validade é dada de dado a dado, ou seja, um dado X pode não possuir o mesmo período de validade que um dado Y, mesmo que ambos tenham sido criados ao mesmo tempo (onde X é diferente de Y).

Para muitos, a estatísticas não passa de conjuntos de tabelas de dados numéricos, onde os estatísticos são aqueles que coletam esses dados. Porém, essa estatística descritiva vai além deste conceito, como citado acima. Sendo assim, pode ser tida como um conjunto de técnicas que permite, de forma sistemática, organizar, descrever, analisar e interpretar dados oriundos de estudos ou experimentos. Diante disso, a estatística descritiva é a etapa inicial da análise utiizada para descrever e reumir os dados.

Uma área muito importante nas aplicações deste conteúdo, é a de amostragem, que se dá na análise de dados afim de obter um novo dado em comum, como por exemplo, pesquisas de opiniões e avaliações, onde formulários são gerados a partir de dados construídos por terceiros. esses formulários são posteriormente analisados e geram um novo dado a partir destes. Diante disso, existêm 4 principais métodos de realização de amostragem, sendo eles:

  • Aleatória

    • Onde cada elemento de um conjunto tem a mesma chance de ser escolhida, ou seja, em um conjunto de N elementos, cada elemento possui exatamente a mesma probabilidade de ser optado.

  • Estratificada

    • Se baseia em classificar o conjunto categorias e extrair um dado de cada categoria, ou seja, por exemplo, em um conjunto população, existem 2 categorias (Homens e Mulheres), então será extraído os dados com base em 1 elemento de cada categoria.

  • Sistemática

    • É constituído em um conjunto de N elementos separados em conjuntos menores de ordem K, ou seja, em um conjunto de 10 elementos por exemplo, será sub-dividido em conjuntos menores tomados de 2 a 2, com o objetivo de diminuir a complexidade da estatística, uma vez que esses elementos estejam ordenados.

  • Conglomerados

    • Se consiste em dividir em seções a área populacional, selecionar aleatoriamente algumas dessas seções e tomar todos os elementos das mesmas

Ou seja, a estatística descritiva se baseia em adquirir um conjunto de dados brutos, realizar cálculos em cima desses dados a fim de obter novos dados, para assim por gerar formas de exibição desse novo conjunto obtido, como gráficos de barra e gráficos pizza. Diante disso, existêm operaçõesq ue se fazem de extrema importância na obtenção desses conjuntos, sendo eles:

  • Média - Cálculo em que sepode obter uma relação entre todos os elementos de um conjunto:
    •             Média = ∑ elementos / Quantidade de elementos
            
        Por exemplo:
        
            Conjunto = {3,43,8643,4,2,0,0.52,250}
            
            Média = 8945.52 / 8 = 1118.19
  • Moda - A moda se dá pelo elemento do conjunto que mais se repete (Hásitações em quenão existem modas ou até mesmo que existem mias de 1 moda):
    •         Por exemplo:
            
            Conjunto = {2,5,7,8,6,6,6,4,2,22,23,4,5}
            
            Moda = 6 (3x)
  • Mediana - Elemento central entre todos os do conjunto (Há situações em que a quanidade de elementos do conjunto é par, ou seja, não haverá um valor central, mas 2. Nesta sitação, basta realizar o cálculo da média entre esses 2 elementos):
    •         Por exemplo:
        
            Conjunto = {1,2,3,4,5}
            
            Mediana = 3
